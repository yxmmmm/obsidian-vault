# include<string.h>
# 1字符串连接函数strcat(dest,src)
| dest | src       |
| ---- | --------- |
| 数组名  | 数组名/字符串常量 |
- src连接dest后src第一个字符将覆盖dest的\0
- dest留有足够空间
例
```c
char dest[20]={"str"};
char src[20]={"cat"};
strcat(dest,src);
printf("%s\n",dest)//输出strcat
srtcat(dest," string")//src位置为字符串常量
printf("%s\n",dest)//输出strcat string
```
# 字符串拷贝函数strcpy(dest,src)
| dest | src       |
| ---- | --------- |
| 数组名  | 数组名/字符串常量 |
- 将src拷贝到dest中
- 连同\0一起拷贝
- ```c
  dest s t r i n g \0 \0
  src  c o p y \0
  dest c o p y \0 g \0 \0
  //g无法输出
  ```
# 字符串比较函数strcmp（str1,str2)
| str1    | str2    |
| ------- | ------- |
| 数组名/字符串 | 数组名/字符串 |
- 逐个比较str1和str2
- str1=str2，函数值为0
- str1< str2，函数值为正
- str1> str2,  函数值为负
- 实际比较字符ASCII
- 出现不同字符就结束

- str可为字符串常量

# strcspn
### 1. **去掉 fgets 的换行符**（最常用）
```c
char input[100];
fgets(input, sizeof(input), stdin);
input[strcspn(input, "\n")] = '\0';  // 去掉换行符
```
### 2. **查找特定字符位置**
```c
char email[] = "user@example.com";
int at_pos = strcspn(email, "@");  // 查找@符号位置
printf("@在位置: %d\n", at_pos);  // 输出: 4
```
