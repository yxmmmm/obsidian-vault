# 输出
```c
char str[10]{"abcd\0efgh"};

a b c d \0 e f g h \0

```
puts(str)打印字符数组str和换行
# 输入
```c
char str[10];
scanf("%s",str);//参数都用数组名
假设输入\0abcd\0efgh
输出结果abcd\0 注：这里的\0是结束标志符
```
非空格开始，第一个空格结束
gets(str)输入字符串给数组str直到遇到换行符或文件结束符
fgets(str，LEN+1,stdin)防止溢出
- 读取len个字符放入str
- 读取换行符，区别gets，读取结束会在末尾添加‘\0’
