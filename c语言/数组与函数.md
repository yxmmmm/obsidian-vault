==形参数组和实参数组共用一段内存==
1. 数组名做函数参数，必须**定义**数组
2. 实参和形参数组的**类型**必须一致
3. 实参和形参数组的大小可一致或不一致
4. 形参数组不可指定大小，**另设一参数**，**传递数组元素的个数
5. 数组名做实参，是把**首地址**传给形参，==**实参形参共用一段内存单元**==
## eg选择排序
```c
#include<stdio.h>
void sort(int x[ ], int n){
	int i, j, t;
	for (i = 0; i <= n-2; i++)
		for (j = i+1; j <= n-1; j++){
			if (x[i] > x[j]){
				t = x[i];
				x[i] = x[j];
				x[j] = t;
			}
		}
}//形参实参共用内存单元，无需返回值

int main(void)
{
	int a[5] = {9,8,7,6,5};
	int b[6] = {11,9,7,5,3,1}
	int i, size = 5;
	
	sort(a, size);
	sort(b, size+1);
	sort(&b[1], size+1)//从b[1]开始到b[3]结束
	
	for(i = 0; i <= size; i++)
		printf("%d", b[i]);
	return 0;
}
```
# 多维数组与函数
1. 可作实参或形参
2. 作形参时，定义时可指定每一维大小，也可省略第一维大小的说明
3. 数组的大小需要另设参数来传递
## eg转置函数
```c
#include<stdio.h>
#define M 2
#define N 3

//转置函数trans
void trans(int x[][N], int y[][M], int m, int n){
	int i,j;
	for(i = 0; i <= m-1; i++){
		for(j = 0; j <= n-1; j++)
			y[j][i] = x[i][j];
	}
}
int main(void){
	int a[M][N] = {{1,2,3}, {4,5,6}}, b[N][M], i, j;
	
	//调用转置函数
	trans(a, b, M, N);
	
	for(i = 0; i <= N-1; i++){
		for(j = 0; j <= M-1; j++)
			printf("%d ", b[i][j]);
		printf("\n");
	}
	return 0;
}
```
