# 局部变量
作用域：定义开始到函数结束
不同函数可用相同变量
# 全局变量
不属于任何一个函数
- 首字母大写
- 默认初始化为0
- 尽量少用，占内存，降低通用型和清晰性
- 若同名，在局部变量范围内，全局变量被屏蔽
