```c
int main(void){
float a=5,b=6,c;
float avg(float a,float b);//函数声明不占内存，可省略参数名
c=avg(a,b);
printf("%f\n",c);
}
float avg(float x,float y){
return (x+y)/2;
}//函数定义开辟内存空间
```
==被调函数出现在主调函数之前，可不用声明==
## 注：函数外部做声明，函数内部无需声明,函数顺序不影响
```c
#include<stdio.h>

float avg(float ,float);//外部声明

float avg(float x,float y){
return (x+y)/2;
}
int main(void){
float a=5,b=6,c;
c=avg(a,b);
printf("%f\n",c);
return 0;
}
```
